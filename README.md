# TrapkaPlugin

Плагин для Paper/Spigot 1.21.8: предмет "Трапка" создает временную комнату вокруг игрока.

## Что умеет

- `/trapka give [игрок] [кол-во]` - выдать трапку.
- `/trapka help` - показать помощь.
- `/trapka reload` - перезагрузить `config.yml`.
- Подсказки команды для количества: `1`, `16`, `32`, `64`.
- ПКМ предметом создает комнату с внутренним пространством 3x3x3.
- Через 10 секунд все замененные блоки восстанавливаются.
- Пока трапка активна, ее блоки нельзя сломать.
- Игрок внутри активной трапки не может ломать блоки снаружи через досягаемость.
- Внутри ставится световой блок.
- Оболочка строится из обычного чернита, полированного чернита и максимум 2 случайных ступенек из полированного чернита.
- Размеры, длительность, материалы и сообщения меняются в `plugins/TrapkaPlugin/config.yml`.

## Сборка через GitHub

1. Залей все файлы проекта в репозиторий GitHub.
2. Открой вкладку `Actions`.
3. Запусти workflow `Build` или просто сделай push.
4. Готовый `.jar` появится в artifacts сборки.

## Локальная сборка

Нужны Java 21 и Maven:

```bash
mvn package
```

Готовый файл будет в папке `target`.
